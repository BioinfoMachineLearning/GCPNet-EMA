What is GCPNet-EMA?

GCPNet-EMA is an application that allows you to predict per-residue lDDT scores for computationally-predicted (e.g., AlphaFold 2) protein structure decoys using GCPNet, our newly-developed graph neural network for learning efficiently from 3D biomolecular structures.
